Image Search Abstraction Layer

by Andres Talero

Call the API at https://image-search-andres.glitch.me/<call>,

where <call> is one of the following:

  • imagesearch/<searchterm>, where <searchterm> is your search keywords (spaces are valid!)
  • recentsearches to view all previous searches made using this microservice

You may add the page parameter to switch pages, if it is not added to the URL then you are on page 1. There are 10 results per page. Examples:

  • https://image-search-andres.glitch.me/imagesearch/bunnies?page1   => first page of bunnies images (first ten results)
  • https://image-search-andres.glitch.me/imagesearch/bunnies?page2   => second page of bunnies images (results 11 to 20)
  • https://image-search-andres.glitch.me/imagesearch/bunnies?page3   => third page of bunnies images (results 21 to 30)
